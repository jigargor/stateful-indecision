# Improving Autonomous AI Agents with Reflective Tree Search and Self-Learning

Source URL: https://arxiv.org/html/2410.02052v1
Source ID: 2410.02052v1
Title: Improving Autonomous AI Agents with Reflective Tree Search and Self-Learning
Authors: Xiao Yu, Baolin Peng, Vineeth Vajipey, Hao Cheng, Michel Galley, Jianfeng Gao, Zhou Yu

## Abstract

Autonomous agents have demonstrated significant potential in automating complex multistep decision-making tasks. However, even state-of-the-art vision-language models such as GPT-4o remain below human-level performance in intricate web environments and long-horizon planning tasks. The paper introduces Reflective Monte Carlo Tree Search (R-MCTS), a test-time algorithm for improving agent exploration. R-MCTS extends MCTS with contrastive reflection, so agents learn from past interactions during search, and multi-agent debate, so state evaluation is more reliable. The paper also studies self-learning by fine-tuning GPT-4o on tree traversals generated by R-MCTS without human-provided labels. On VisualWebArena, GPT-4o with R-MCTS improves by 6% to 30% relative across tasks; fine-tuned GPT-4o recovers 97% of R-MCTS performance while reducing test-time compute by a factor of four.

## Research Hooks For This Run

- Treat R-MCTS as a candidate mechanism for improving the beta agents' learning rate: convert search traces, checker critiques, and accepted memory writes into future policy/retrieval tuning signals.
- Compare R-MCTS with the Auton paper's factorized policy, memory consolidation, and self-evolution loop.
- Track whether multi-agent debate improves checker confidence and reduces unsupported claims in this repo's three-agent research loop.
- Extract concrete learning-rate signals: better seed selection, better action-vocabulary weights, better retrieval scope, and stronger validation before memory writes.
